Prontera To-do
==============

### 개발 예정 기능

+ 태그 목록 지원
+ 방명록 페이지 지원
+ 웹 접근성 개선
+ 읽기 모드에서의 맥락 안정화
+ 웹 표준 개선
+ 브라우저 테스트
+ 디자인 통일성 개선
+ No-js 환경 테스트
+ No-css 환경 테스트
+ 리스트 스타일


### 생각 중인 기능

+ 갤러리 스타일의 게시글 리스트
+ 사이드바의 최근 댓글 패널
+ 영어 지원
+ Google Analytics 지원
+ 동일 카테고리 다른 게시글 표시
+ 티스토리 제공 Summary 대신 원본 게시글 사용 `연구 필요`
